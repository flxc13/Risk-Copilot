# Risk Copilot

Production-style portfolio risk platform built in phases.

## Phase Plan
- Phase 1: Core portfolio + market data + risk metrics + UI baseline
- Phase 2: AI Risk Copilot for risk explanations and what-if analysis
- Phase 3: AWS deployment hardening

## Quick Start
1. Create and activate a virtual environment.
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Run API:
   - `uvicorn app.api.main:app --reload`
4. Run tests:
   - `pytest -q`
